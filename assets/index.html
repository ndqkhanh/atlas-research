Atlas-Research

Long-horizon research-report agent with planner + retriever + mandatory citation verifier.

Design target: ≥95% faithfulness (claim-citation-bound outputs) with bounded per-report cost. Current: MVP with mock retriever; pipeline is real, retrieval is stubbed.

What Atlas-Research is

Planner (ReWOO-style DAG) → Source Router → Retrieval Executor → Synthesis Writer → Citation Verifier. Every claim in the final report must bind to evidence from the trace; unverified claims get flagged or dropped.

Status

MVP. 13/13 tests passing. Mock retriever ships canned results.

Stack

Python; FastAPI; harness_core.
